# Crypto Trader — Chat Module
## AI-Powered Chatbot with MCP Tool Integration

---

Crypto Trader Chat is an AI‑driven application that provides an intelligent
interface for users and admins to interact with the Crypto Trader platform. It
functions as an **MCP Client**, orchestrating conversations with an LLM
(such as OpenAI) while dynamically leveraging tools provided by the
`Crypto‑Trader‑Agent` MCP server.
