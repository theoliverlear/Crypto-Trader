This is a placeholder page to satisfy MkDocs strict mode when aggregated Javadoc hasn't been generated yet.

During CI, the Maven build runs maven-javadoc-plugin:aggregate and copies the output from target/site/apidocs into this folder. When that happens, this page will be replaced with the real Javadoc index.

If you're viewing this locally, run the following at the repository root before building the docs:

mvn -B -DskipTests verify
