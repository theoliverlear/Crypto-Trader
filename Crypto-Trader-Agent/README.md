# Crypto Trader — Agent Module
## Model Context Protocol (MCP) Server with Crypto Trader Domain Knowledge

---

Crypto‑Trader‑Agent is a specialized Spring Boot application that acts as a
**Model Context Protocol (MCP) Server**. It exposes the vast library of
Crypto Trader services as standardized tools that can be called by LLMs and AI
assistants.
